## Overview

Lidger is a personal dashboard designed to provide a single source of truth for goal tracking and personal development. Built for the way I actually want to manage my life.

## Key Features

### Goal Management (OKRs)
- Set quarterly and annual objectives
- Define measurable key results
- Track progress with visual indicators
- Review and adjust goals regularly

### Task Management
- Priority matrix (urgent/important quadrants)
- Daily and weekly planning views
- Recurring task support
- Integration with OKRs for alignment

### Health Tracking
- Weight and body composition
- Exercise and activity logs
- Sleep quality metrics
- Nutrition tracking

### Habit Tracking
- Daily habit streaks
- Habit stacking support
- Visual streak calendars
- Trend analysis over time

## Technical Highlights

Phoenix monolith with rich interactivity:
- LiveView for real-time updates without SPA complexity
- Chart.js for data visualization
- Mobile-responsive design for on-the-go logging
- Data export for analysis
